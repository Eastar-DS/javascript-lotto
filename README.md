# TDD기반 로또 페어 프로그래밍

## 한 줄 요약

구매 로또와 정답 로또를 비교해서 정답 개수 세기

## 기능 목록

### 1. 로또 구입

- 로또 구입 금액을 입력받는 기능
  - [예외] 빈 값인 경우
  - [예외] 문자가 포함된 경우
  - [예외] 1,000원 단위가 아닌 경우
  - [예외] 0이 입력된 경우
  - 예외 발생 시 해당 부분부터 다시 입력받음

### 2. 로또 발행

- 발행해야하는 로또 개수를 계산하는 기능
- 구입 수량만큼 로또 번호를 무작위로 6개씩 생성하는 기능
  - [예외] 0또는 음의 정수인 경우
  - [예외] 1~45 사이가 아닌 경우
  - [예외] 로또 번호가 중복되는 경우
  - [예외] 로또 번호가 6개가 아닌 경우
  - 예외 발생 시 해당 로또 번호에 대해서만 다시 생성
- 생성된 번호를 오름차순 정렬하는 기능
- 구입 수량만큼 로또를 발행하는 기능

### 3. 당첨 로또 발행

- 당첨 번호를 입력받는 기능
  - [예외] 빈 값인 경우
  - [예외] 쉼표 이외의 문자가 포함된 경우
  - [예외] 당첨 번호가 0또는 음의 정수인 경우
  - [예외] 당첨 번호가 1~45 사이가 아닌 경우
  - [예외] 당첨 번호가 중복되는 경우
  - [예외] 당첨 번호가 6개가 아닌 경우
  - 입력값에 공백이 포함되어도 예외로 처리하지 않음
  - 예외 발생 시 해당 부분부터 다시 입력받음

- 보너스 번호를 입력받는 기능
  - [예외] 빈 값인 경우
  - [예외] 문자가 포함된 경우
  - [예외] 보너스 번호가 0또는 음의 정수인 경우
  - [예외] 보너스 번호가 1~45 사이가 아닌 경우
  - [예외] 보너스 번호가 당첨 번호와 중복되는 경우
  - 예외 발생 시 해당 부분부터 다시 입력받음

### 4. 당첨 여부 확인

- 구매한 모든 로또의 등수를 계산하는 기능
- 수익률을 계산하는 기능
  - 소숫점 둘째자리에서 반올림

```
1등: 6개 번호 일치 / 2,000,000,000원
2등: 5개 번호 + 보너스 번호 일치 / 30,000,000원
3등: 5개 번호 일치 / 1,500,000원
4등: 4개 번호 일치 / 50,000원
5등: 3개 번호 일치 / 5,000원
```

### 5. 게임 종료 및 재시작

- 재시작 여부를 입력받는 기능
  - [예외] y 또는 n이 아닌 경우 (대문자 허용)
  - 예외 발생 시 해당 부분부터 다시 입력받음
